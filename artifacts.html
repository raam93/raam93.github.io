RaR: Reasoning About Reasoning

Paper: Reasoning About Reasoning: Towards Informed and Reflective Use of LLM Reasoning in HCI

A structured worksheet that supports critical reflection on how LLM reasoning is identified, framed, executed, and evaluated in HCI research and practice.

Worksheet

ArC: Argument-Based Consistency

Paper: Argument-Based Consistency in Toxicity Explanations of LLMs (EACL '26)

A codebase with a suite of metrics for assessing multiple forms of logical consistency in LLM-generated free-form explanations of harmful content.

Code

AsAP: Assumption As Premise

Paper: Talking About the Assumption in the Room (CHI '25)

A methodological framework that formalizes the articulation of implicit assumptions in AI workflows, making them explicit, inspectable, and contestable.

Framework

Non-Ideal Responsible ML Framework

Paper: Towards a Non-Ideal Methodological Framework for Responsible ML (CHI '24)

A methodological framework and visualizer for systematically mapping abstract ethical values and concrete technical interventions in AI.

Framework Visualizer

Codebase to generate a set of counterfactual explanations for any Machine Learning model.
